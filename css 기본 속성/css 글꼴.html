글꼴

font-family

하나의 글꼴만을 설정할 수도 있고, 여러 개의 글꼴을 같이 설정할 수 있음
첫 번째 글꼴이 사용자의 컴퓨터에 없으면 다음 글꼴을 확인한다


font-style

주로 이탤릭체를 사용하기 위해 사용한다. 3가지의 속성값이 있음
font-style: normal
font-style: italic
font-style: oblique


font-variant

속성값 small-caps는 소문자를 작은 대문자로 변경시킴
small-caps


font-weight

텍스트의 두께를 설정함
font-weight: 500
font-weight: border


font-size

텍스트의 크기를 설정
font-size: 200%
font-size: 3em
font-size: 50px
