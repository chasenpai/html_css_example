블록(block)타입 요소

display 속성값이 블록(block)인 요소는 항상 새로운 라인에서 시작하며, 해당 라인의 모든 너비를 차지한다.


<div>태그

div는 여러 요소들을 하나로 묶는데 자주 사용되는 대표적인 블록요소이다.

div로 여러 요소들을 묶으면 한번에 CSS 스타일을 적용시킬 수 있다.


인라인(inline)타입 요소

display 속성값이 인라인(inline)인 요소는 새로운 라인에서 시작하지 않으며
요소의 너비가 해당 HTML 요소의 내용만큼 차지한다.
span 태그는 대표적인 인라인요소로, 주로 텍스트의 특정 부분을 묶어서
따로 스타일을 적용하기 위해 사용된다.
